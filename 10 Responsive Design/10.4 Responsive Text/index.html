Pixel Value (px)

This text is set with a pixel value, which does not scale with the user's settings or the device's resolution.

Em Value (em)

The em value scales relative to the font size of its closest parent element.

Rem Value (rem)

The rem value scales relative to the root element's font size, typically the <html> element.

Viewport Unit (vw)

The viewport unit (vw) scales based on the width of the viewport.

Media Query

Media queries allow us to conditionally apply styles based on device characteristics, such as its width.

Clamp()

The clamp() function offers a way to set responsive typography with minimum, preferred, and maximum values.
